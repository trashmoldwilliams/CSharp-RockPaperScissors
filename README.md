# Rock Paper Scissors

#### By Will Johnson

## Description

A C# assignment to create a rock paper scissors app. The concept is generally simple and the main purpose is for practicing Xunit testing.

## Setup/Installation Requirements

* Windows Machine
* Powershell

* Clone Project
* Run DNU Restore
* Run DNX Kestrel
